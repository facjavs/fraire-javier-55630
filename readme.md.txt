Descripción del Proyecto:
Página Web para la compra y venta de piezas artisticas

Al estar en la pagina de inicio, los usuarios podran registrarse para poder tener acceso a las demas funcionalidades de la pagina, pudiendo asi realizar las siguientes acciones:
Publicar un producto para su venta
Usar la barra de busqueda para encontrar un articulo en especial 
Navegar por el catalogo de productos
